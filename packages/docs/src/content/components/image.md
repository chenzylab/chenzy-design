## 使用场景

Image 是增强型图片展示组件，用于替代原生 `<img>` 标签。在原生图片基础上，提供加载占位、加载失败降级、懒加载、预览放大（lightbox）等能力。适合产品图片展示、内容配图、用户上传图片预览、图片画廊等场景。

Image 支持单图点击放大预览与 ImagePreviewGroup 多图预览联动，预览模式下支持缩放、旋转、左右切换，提供沉浸式的图片查看体验。

## 何时使用

在需要展示图片并可能需要用户放大查看细节的场景中使用 Image。相比原生 `<img>`，Image 提供了更完善的加载状态管理与无障碍支持。如果只需要展示纯装饰图（不需要可访问名称），使用原生 `<img alt="">` 即可。

## 无障碍

- 所有图片必须提供 `alt` 属性；纯装饰图使用 `alt=""`，内容图使用描述性文案。
- 可点击进入预览的图片包裹元素使用 `role="button"` + `tabindex="0"`，支持 Tab 聚焦与 Enter/Space 键触发预览。
- 预览 lightbox 使用 `role="dialog"` + `aria-modal="true"`，并通过 `useFocusTrap` 限制焦点在弹层内循环。
- 预览键盘交互：Esc 关闭，←/→ 切换上一张/下一张，+/- 缩放，0 重置；关闭后焦点归还触发元素。
